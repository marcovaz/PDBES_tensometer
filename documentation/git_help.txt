QUICK COMMANDS TO START
- To check out a working copy of an existing repository in another machine
	git clone ssh://git@github.com:marcovaz/PDBES_tensometer.git
- To communicate with the master repository: git pull to get the changes, git push to commit them onto the master
	git push -u origin master
	git pull -u origin master
- To add a new file in the repository
	git add myfile
- To edit a file content in the repository
	git checkout myfile
- To commit the changes into your own local repository
	git commit -m "my commit message"
- To check which files have been changed, which files are to be committed, etc
	git status
